    • delete(
          body: Tags_resource,
          requestConfiguration?: RequestConfiguration<object>,
      ): Promise<void>

      Resources can be untagged by sending a DELETE request to/v2/tags/$TAG_NAME/resources with an array of json objects containingresource_id and resource_type attributes.Currently only untagging of Droplets, Databases, Images, Volumes, and VolumeSnapshots is supported. resource_type is expected to be the string droplet,database, image, volume or volume_snapshot. resource_id is expectedto be the ID of the resource as a string.In order to untag a resource, you must have both tag:delete and <resource type>:update scopes. For example, to untag a Droplet, you must have tag:delete and droplet:update.
